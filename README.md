# Веб-сервис учета и обработки заявок на ИТ-обслуживание для ООО «Контроль ИТ»

Проект подготовлен для преддипломной практики по направлению 38.03.05 «Бизнес-информатика», профиль «Цифровая экономика».

## Назначение

Веб-сервис автоматизирует бизнес-процесс обработки заявок клиентов на ИТ-обслуживание и контроль выполнения работ. Решение позволяет клиентам создавать обращения, специалистам обрабатывать заявки, а администратору управлять пользователями, услугами, справочниками и отчетами.

## Стек

- Python 3.9+
- Flask
- HTML-шаблоны Jinja2
- CSS и JavaScript
- SQLite
- SQLAlchemy
- Flask-Login
- python-docx для формирования актов
- openpyxl для формирования отчетов Excel

## Запуск

```bash
python -m venv .venv
.venv\Scripts\activate       # Windows
# source .venv/bin/activate   # macOS/Linux
pip install -r requirements.txt
python main.py
```

После запуска сайт будет доступен по адресу:

```text
http://127.0.0.1:5000
```

База данных создается автоматически при первом запуске в папке `data/`.

## Соответствие требованиям индивидуального задания

Проект реализован как сетевой ресурс на Python + Flask, содержит публичные страницы, личные кабинеты, административную панель, разграничение прав доступа, SQL-базу данных, загрузку файлов, формирование документов `.docx` и отчетов `.xlsx`.

Дополнительные материалы для проверки находятся в папке `docs/`: техническое задание, чек-лист требований к сетевому ресурсу, чек-лист требований к ПО и руководство пользователя. Исполняемый файл `.exe` не формируется, запуск выполняется через `python main.py`.
