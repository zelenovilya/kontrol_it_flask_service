# Чек-лист требований к сетевому ресурсу

| Требование | Реализация |
|---|---|
| Python + Flask | Используется Flask-приложение |
| HTML-страницы | Шаблоны Jinja2 в папке `app/templates` |
| База данных SQL | SQLite + SQLAlchemy |
| Не менее 10 таблиц | 14 моделей данных |
| Не менее 10 публичных страниц | Главная, о компании, услуги, тарифы, инструкция, преимущества, FAQ, база знаний, новости, контакты, обратная связь, вход, регистрация |
| Роли пользователей | Администратор, специалист, клиент |
| Админ-панель | 10 разделов управления |
| Управление пользователями | Добавление, редактирование и блокировка учетных записей |
| Личные кабинеты | Кабинет клиента и кабинет специалиста |
| Загрузка файлов | Прикрепление файлов к заявкам |
| Формирование документов | Акт выполненных работ `.docx` |
| Формирование отчетов | Отчет по заявкам `.xlsx` |
| Хлебные крошки | Есть в базовом шаблоне |
| Подвал с ФИО автора | Есть в базовом шаблоне |
| requirements.txt | Есть в корне проекта |
| main.py | Есть в корне проекта |
