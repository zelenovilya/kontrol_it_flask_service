# Руководство пользователя

## Запуск

1. Установить Python.
2. Перейти в папку проекта.
3. Создать виртуальное окружение и установить зависимости:

```bash
python -m venv .venv
pip install -r requirements.txt
```

4. Запустить приложение:

```bash
python main.py
```

5. Открыть сайт в браузере:

```text
http://127.0.0.1:5000
```

Если порт `5000` занят, приложение можно запустить на другом порту через Flask или изменить порт в команде запуска.

## Тестовые учетные записи

| Роль | Логин | Пароль |
|---|---|---|
| Администратор | `admin` | `admin123` |
| Специалист | `specialist` | `spec123` |
| Клиент | `client` | `client123` |

## Клиент

Клиент может зарегистрироваться, войти в личный кабинет, создать заявку, выбрать услугу и приоритет, прикрепить файл, просматривать свои заявки, добавлять комментарии и скачивать сформированные документы.

## Специалист

Специалист видит назначенные и новые заявки, открывает карточку заявки, меняет статус, назначает исполнителя, добавляет комментарии и файлы, просматривает историю работ, формирует акт выполненных работ в формате `.docx` и скачивает отчет `.xlsx`.

## Администратор

Администратор управляет пользователями, ролями, клиентами, сотрудниками, услугами, статусами заявок, новостями, базой знаний, обращениями обратной связи и отчетами. В разделе пользователей доступны добавление, редактирование и блокировка учетных записей.
