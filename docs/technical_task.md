# Краткое техническое задание

## Наименование

Веб-сервис учета и обработки заявок на ИТ-обслуживание для ООО «Контроль ИТ».

## Цель создания

Автоматизировать бизнес-процесс приема, регистрации, распределения, выполнения и контроля заявок клиентов на ИТ-обслуживание.

## Пользователи

- Администратор.
- Специалист ООО «Контроль ИТ».
- Клиент.

## Основные функции

- регистрация и авторизация пользователей;
- создание заявок клиентами;
- назначение заявок специалистам;
- изменение статусов заявок;
- ведение комментариев и истории обработки;
- загрузка файлов к заявкам;
- формирование акта выполненных работ в формате `.docx`;
- формирование отчета по заявкам в формате `.xlsx`;
- управление услугами, пользователями, статьями и обращениями обратной связи.

## Требования к данным

База данных должна содержать сведения о пользователях, ролях, клиентах, сотрудниках, услугах, заявках, статусах, приоритетах, комментариях, файлах, документах, обращениях обратной связи и материалах базы знаний.
