# Подсчет логических строк проекта

Подсчет выполнен для файлов `.py`, `.html`, `.css`, `.js` по скрипту из проверочного чек-листа.

Не учитывались:

- пустые строки;
- строки комментариев;
- строки, состоящие только из одиночных управляющих символов или закрывающих тегов.

Итоговый объем после доработки:

| Тип файлов | Логических строк |
|---|---:|
| Python | 1657 |
| HTML | 2314 |
| CSS | 322 |
| JavaScript | 7 |
| Всего | 4300 |

Требование объема более 2000 логических строк выполнено с запасом. Прирост получен за счет содержательных публичных страниц, расширенных HTML-шаблонов, закрытых кабинетов, административных разделов, маршрутов скачивания файлов, seed-данных и служебной документации проекта.
